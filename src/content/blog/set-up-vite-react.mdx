## 2. Configure ESLint

[ğŸ“˜ ESLint Getting Started](https://eslint.org/docs/latest/use/getting-started#quick-start)

Run the ESLint init command:

```bash
npm init @eslint/config@latest

```

Select the following options:

- Language: `JavaScript`
- Style: `problems`
- Modules: `ESM`
- Framework: `React`
- Use TypeScript: `Yes`
- Environment: `Browser`

Accept all suggested dependencies and install with `npm`.

### Add to `eslint.config.js`:

```js
{
  files: ['**/*.{js,mjs,cjs,ts,jsx,tsx}'],
  languageOptions: { globals: globals.browser },
  settings: {
    react: {
      version: 'detect',
    },
  },
}

```

## 3. Setup Prettier

[ğŸ“˜ Prettier + ESLint Integration](https://github.com/prettier/eslint-plugin-prettier)

Install:

```bash
npm install --save-dev eslint-plugin-prettier eslint-config-prettier
npm install --save-dev --save-exact prettier

```

Create `.prettierrc`:

```json
{
  "singleQuote": true,
  "semi": false,
  "tabWidth": 2,
  "trailingComma": "es5",
  "printWidth": 100
}
```

Create `.prettierignore`:

```text
node_modules
dist
build
*.min.js

```

### Update `eslint.config.js`:

Make sure to add `eslint-plugin-prettier/recommended` as the last item in the config array:

```js
export default defineConfig([
  {
    files: ["**/*.{js,mjs,cjs,ts,jsx,tsx}"],
    plugins: { js },
    extends: ["js/recommended"],
  },
  {
    files: ["**/*.{js,mjs,cjs,ts,jsx,tsx}"],
    languageOptions: { globals: globals.browser },
    settings: { react: { version: "detect" } },
  },
  tseslint.configs.recommended,
  pluginReact.configs.flat.recommended,
  eslintPluginPrettierRecommended, // Must be last
]);
```

## 5. Husky, Lint-Staged, and Commitlint

### Setup Husky

[ğŸ“˜ Husky Docs](https://github.com/typicode/husky/blob/main/docs/get-started.md)


### Install Lint-Staged

[ğŸ“˜ Lint-Staged Docs](https://github.com/lint-staged/lint-staged)


Add to `package.json`:

```json

  "**/*.{js,ts,jsx,tsx}": [

    "prettier --write"
  ]
```

### Install Commitlint

[ğŸ“˜ Commitlint Docs](https://commitlint.js.org/)

Create `commitlint.config.js`:

```js

  extends: ["@commitlint/config-conventional"],
  helpUrl:
    "https://github.com/conventional-changelog/commitlint/#what-is-commitlint",
};

export default Configuration;
```

### Husky Hooks

**`.husky/commit-msg`**:

```bash

. "$(dirname "$0")/_/husky.sh"

npx --no-install commitlint --edit "$1"

```

## 6. Configure Vitest

[ğŸ“˜ Vitest Docs](https://vitest.dev/)

```bash
npm install -D vitest @vitest/coverage-v8

```

Create `vitest.config.ts`:

```tsx
import { defineConfig } from "vitest/config";

export default defineConfig({
  test: {
    coverage: {
      include: ["src/**/*.{ts,tsx,js,jsx}"],
      exclude: ["**/*.test.*", "**/__tests__/**"],
      reporter: ["text", "html"],
    },
  },
});
```

Update `package.json`:

```json
"scripts": {
  "test": "vitest",
  "coverage": "vitest run --coverage"
}

```

Update ESLint config to ignore `coverage/`:

```tsx
export default defineConfig([
  { ignores: ['coverage/**'] },
  ...
])

```

## 7. ESLint Tips

To **disable a rule inline**, use:

```tsx
// eslint-disable-next-line no-console
console.log("Debug message");
```

Or globally in `eslint.config.js`:

```js
rules: {
  'no-console': 0
}

```
